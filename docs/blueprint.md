# **App Name**: Attendance Hero

## Core Features:

- Dashboard Selection: Home page to select between a 'General Dashboard' with all offices or a specific 'Office Dashboard'.
- Drag and Drop Attendance: Drag and drop interface to move employee cards between attendance status columns (Present, Absent, Late).
- Office Edit Modal: Modal to edit the office assigned to each employee.
- Attendance Report: Display daily attendance report for each office, summarizing the number of present, absent, and late employees.
- Copy Report Button: Button to copy the daily attendance report text.

## Style Guidelines:

- Primary color: Soft blue (#7BB4E5), evokes trust and efficiency.
- Background color: Light gray (#F2F4F7), for a clean, non-distracting workspace.
- Accent color: Light orange (#E5A07B) to draw attention to interactive elements and key data.
- Font: 'Inter', a sans-serif font for clean readability in dashboards and reports.
- Use clear and simple icons to represent attendance status and office locations.
- Dashboard layout should be clean and intuitive, using card-based design for employees and offices.
- Use subtle transition animations for drag-and-drop actions and modal appearances.
